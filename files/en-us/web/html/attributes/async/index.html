The async attribute is a boolean type of attribute which indcates browser how script should be executed.

The async attribute is an allowed attribute for {{htmlelement("script")}}. When used on a <script> element, the script will be executed automatically without clicking on any button, with loading of page.

Note: Insted you can use defer which will execute the script when page has finished parsing.

Usage

The async can be used on <script> element.

Note: Use the async attribute only when script is external file and linked to web page by using src attribute.

Syntax

<script src="sample_async.js" async></script>

Examples

  <form accept-charset="UTF-8">
    <label for="fname">First name:</label>
    <input type="text" id="fname" name="fname">
    <label for="lname">Last name:</label>
    <input type="text" id="lname" name="lname">
    <label for="email">E-mail ID:</label>
    <input type="email" id="email" name="email">
    <input type="Submit" id="submit" name="submit">
    <input type="reload" id="reload" name="reload">
  </form>
  <script src="demo.js" async>
  </script>

Script file

  alert("For Registration: Please fill and submit the form")

When you will run the code in example, page and script file will load at the same time.So form and pop-up, both will come(load) at same time.

See examples of usage on the element pages for {{htmlelement("script")}}.
